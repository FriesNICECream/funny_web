# Data Model: Stitch Media Data 仪表盘

## 媒体资源项

- **说明**: 页面中展示的核心影视内容单元。
- **字段**:
  - `id`: 唯一标识
  - `title`: 资源标题
  - `subtitle`: 补充说明或简述
  - `coverImage`: 封面图地址
  - `releaseYear`: 年份信息
  - `rating`: 评分值
  - `qualityBadge`: 规格标记，例如 4K、Remux
  - `genres`: 类型标签列表
  - `links`: 资源链接集合
  - `isFeatured`: 是否为优先展示资源
- **校验规则**:
  - `title` 不能为空
  - `links` 至少允许为空数组，但不可为未定义
  - `rating` 若存在，需可被展示为可读评分文本
  - `genres` 超过展示上限时需要可折叠或截断

## 资源链接项

- **说明**: 某个媒体资源下的单条访问入口。
- **字段**:
  - `id`: 唯一标识
  - `label`: 链接名称
  - `detail`: 规格或说明文案
  - `href`: 目标地址
  - `status`: 可用状态，例如可用、失效、待补充
- **校验规则**:
  - `label` 必须存在
  - `href` 缺失时不可作为主操作展示
  - `status` 决定链接是否可点击以及如何显示反馈

## 导航项

- **说明**: 顶部导航和侧边导航中的单个入口。
- **字段**:
  - `id`: 唯一标识
  - `label`: 显示名称
  - `icon`: 图标标识
  - `targetView`: 指向的内容视图
  - `isActive`: 当前是否激活
- **状态转换**:
  - 默认态 -> 激活态：用户进入对应页面或主动切换
  - 激活态 -> 默认态：用户切换到其他导航项

## 筛选状态

- **说明**: 控制资源列表范围和排序方式的当前状态集合。
- **字段**:
  - `keyword`: 搜索关键词
  - `category`: 当前分类
  - `sortBy`: 当前排序方式
  - `activeChips`: 已生效的筛选条件摘要
  - `resultCount`: 当前结果数量
- **状态转换**:
  - 默认态 -> 已筛选：用户输入关键词或选择筛选项
  - 已筛选 -> 无结果：过滤后无匹配数据
  - 已筛选/无结果 -> 默认态：用户清空条件

## 页面展示状态

- **说明**: 支撑页面反馈与异常场景的聚合状态。
- **字段**:
  - `viewMode`: 桌面或移动展示模式
  - `sidebarOpen`: 移动端侧栏是否展开
  - `isEmpty`: 是否为空列表
  - `hasNoResults`: 是否为无匹配结果
  - `feedbackMessage`: 当前反馈文案
- **状态转换**:
  - 初始态 -> 正常展示：数据加载完成且有内容
  - 初始态 -> 空状态：数据源为空
  - 正常展示 -> 无结果：筛选后无匹配项
  - 移动关闭 -> 移动展开：用户打开侧栏
